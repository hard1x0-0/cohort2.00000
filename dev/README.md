# Development Files

This folder contains various development files and experiments.

## Files

- `hello.cpp` - C++ program
- `main` - Compiled executable (likely from hello.cpp)
- `demo.txt` - Demo text file
- `style1stcalss.css` - CSS stylesheet

## Running the C++ Program

To run the C++ program:

```bash
./main
```

Or recompile if needed:

```bash
g++ hello.cpp -o main
./main
```
